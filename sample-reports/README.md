# Sample Reports

This directory is reserved for sanitized or simulated report examples generated by Windows Enterprise Troubleshooting Framework v2.

## Required labels

Every published example must be labelled as one of:

- `SIMULATED PORTFOLIO DATA`
- `SANITIZED LAB EVIDENCE`
- `SANITIZED PRODUCTION EVIDENCE`

Do not commit unreviewed customer or employer data.

## Recommended examples

- General endpoint health assessment
- DNS and connectivity incident
- Windows Update failure
- Low disk capacity case
- Repeated application crash case
- Before-and-after repair validation

## Expected files per case

```text
CASE-YYYYMMDD-HHMMSS/
    case-metadata.json
    summary.json
    plugin-results.json
    findings.csv
    enterprise-case-report.html
```

The repository includes `sample-data/simulated-case.json` as the first representative data set. A rendered HTML example can be added after the v2 workflow has been executed in a Windows lab.
